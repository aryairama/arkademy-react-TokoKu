## Installation

- Clone This Front End Repo

```
git clone https://github.com/aryairama/arkademy-express-TokoKu.git

```

- Go To Folder Repo

```
cd arkademy-express-TokoKu
```

- Install Module

```
npm install
```

- Type ` npm run start` To Start Website
